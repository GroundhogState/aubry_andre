QETLAB
======

QETLAB is a MATLAB toolbox for exploring and manipulating quantum entanglement.

## Requirements

* MATLAB
* [CVX](http://cvxr.com/cvx/) by CVX Research, Inc.

## Tested with

* MATLAB R2011b
* CVX Version 2.1

## Documentation and Further Information

* [QETLAB homepage](http://www.qetlab.com/)
